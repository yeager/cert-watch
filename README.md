# Cert Watch

TLS/SSL certificate expiry monitor with notifications.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install cert-watch
```

### Fedora/RPM
```bash
sudo dnf install cert-watch
```
